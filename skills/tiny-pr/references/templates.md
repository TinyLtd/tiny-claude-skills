# Release Templates (structure skeletons)

Fill the `{{slots}}`. Auto-insert current boilerplate from `boilerplate.md` (real verbatim Tiny text;
tailor only the FLS topic sentence). Keep Tiny voice (`voice-and-style.md`). Run the lint before staging.

## Common shell (all releases)
```
{{HEADLINE — factual, specific, includes (TSX: TINY) context}}
{{optional sub-head — the one extra fact that matters}}

{{CITY, Province}} — {{Month DD, YYYY}} — Tiny Ltd. ("Tiny" or the "Company") (TSX: TINY) today {{the news in one sentence}}.

{{Body — facts first, most material to least. Numbers with currency. Short paragraphs.}}

{{CEO/CFO quote — attributed, named, opinion not new fact}}

{{Type-specific sections — see below}}

About Tiny
{{ABOUT_TINY boilerplate — current version}}

Forward-Looking Information
{{FLS cautionary note — current counsel-approved version}}

{{TSX exchange disclaimer — current TSX wording, NOT TSX Venture}}

For more information:
{{IR/Media contact — name, title, email, phone}}

SOURCE: Tiny Ltd.
```

## Earnings add-ins
- Selected financial highlights table (revenue, gross profit, Adjusted EBITDA, net income, cash/FCF,
  recurring %), current vs prior period, currency labelled.
- Non-GAAP note: define Adjusted EBITDA and reconcile (or point to MD&A reconciliation).
- Line: "The Company's financial statements and MD&A for {{period}} are available on SEDAR+ at
  sedarplus.ca and on the Company's website."

## Material change / M&A add-ins
- "Transaction Overview": structure, consideration, % acquired, currency.
- "Approvals & Timing": TSX / shareholder / lender / regulatory conditions; expected close.
- If related-party: MI 61-101 paragraph (special committee, valuation, minority approval, who abstained).
- Note the Material Change Report will be filed on SEDAR+ (within 10 days).

## Financing add-ins
- Offering terms: instrument, size, price, currency, underwriters/agents, brokered/bought-deal.
- "Use of Proceeds" (factual).
- "The {{offering}} is subject to TSX approval." + any US/Reg-S selling-restriction legend.
- Closing date.

## Governance / AGM add-ins
- Director change: name, role, effective date, brief bio/board thanks.
- AGM results: per-resolution table (For / Against / Withheld, # and %).
- Dividend: per-share amount, record date, payment date, eligible-dividend designation, currency.

## Company update (non-material) add-ins
- Keep it tight; facts + one quote. Still carries About/FLS/disclaimer/contact/SOURCE.
- Confirm non-materiality before using this lighter template.

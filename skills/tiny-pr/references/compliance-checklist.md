# Compliance Lint — TSX Issuer News Release

Run every item. Mark PASS / FIX / BLOCK. A BLOCK means do not stage.

## Materiality & disclosure basics
- [ ] Release type identified (earnings / material change / financing / governance / company update).
- [ ] Is the information a **material change** (a change in business, operations, capital reasonably
      expected to move the share price)? If yes → news release immediately **and** Material Change
      Report on SEDAR+ within **10 days**. Output the MCR due date.
- [ ] No **selective disclosure** (NP 51-201): every material number/fact here is already public or is
      being filed concurrently. Flag any figure that isn't.
- [ ] Wire-first: confirm the release goes out via the approved wire **before** any social / IR-site /
      email redistribution.

## Earnings-specific
- [ ] Financial statements + MD&A board / Audit-Committee approved (NI 51-102).
- [ ] Statements + MD&A filed on SEDAR+ **concurrently** with this release.
- [ ] Non-GAAP / non-IFRS measures (Adjusted EBITDA, etc.) are **labelled as non-GAAP**, defined, and
      reconciled to the nearest IFRS measure — or the release points to where the reconciliation lives.
- [ ] No new material numbers beyond what the filed statements support.

## Financing / share issuance
- [ ] **TSX conditional approval / pre-clearance** obtained or in process (flag status; many require
      clearance BEFORE the release).
- [ ] If a prospectus/offering: required legends present; no conditioning the market beyond what's
      permitted; US/Reg-S selling-restriction legend if applicable.
- [ ] Use of proceeds stated factually, not promotionally.

## M&A / material transaction
- [ ] If a **related-party transaction** → **MI 61-101** disclosure (special committee, valuation,
      minority approval) referenced as applicable.
- [ ] Closing conditions, approvals (TSX, shareholder, lender), and expected timing stated.
- [ ] No firm assurances of completion; deal described as subject to conditions.

## Forward-looking information (almost always required)
- [ ] FLS cautionary note present (use current counsel-approved version in boilerplate).
- [ ] Forward-looking statements use signal words ("expects", "intends", "anticipates", "may", "will")
      and are tied to the cautionary note.
- [ ] Material assumptions and risk factors referenced (point to MD&A / AIF on SEDAR+).
- [ ] No forward-looking financial outlook (FOFI) unless permitted, dated, and accompanied by the
      required FOFI disclaimer and assumptions.

## Tone / promotional balance
- [ ] Promotional but defensible — no superlatives the facts don't support ("record", "best-ever",
      "transformational") unless literally true and substantiable.
- [ ] Balanced: material risks/qualifiers not buried.
- [ ] Quotes are attributed to a named, authorized spokesperson (CEO/CFO) and are opinion, not new
      material fact introduced only in the quote.

## Mechanics & format
- [ ] Headline + optional sub-head; dateline (City, Province — Date); "(TSX: TINY)" on first mention.
- [ ] Current **TSX** exchange disclaimer (NOT "TSX Venture Exchange").
- [ ] "About Tiny" boilerplate (current version).
- [ ] Contact block with named IR/media contact.
- [ ] "SOURCE: Tiny Ltd." line.
- [ ] Currency stated (CAD vs USD) wherever amounts appear.

## Timing
- [ ] If release time falls in market hours (09:30–16:00 ET) → warn: notify CIRO market surveillance;
      consider a trading halt; prefer pre-open or post-close.

## Final gate
- [ ] Disclosure Committee + CEO approval recorded (name + timestamp).
- [ ] Earnings: board/Audit-Committee approval recorded.
- [ ] All BLOCK items cleared. Only then → stage.
